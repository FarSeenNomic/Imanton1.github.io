🎉 You are invited to play Nomic Kittens 🎉

Nomic is an game where each move in the game changes the rules of the game, even how you win can be modified. This version of Nomic uses Exploding Kittens as a base, so players will be playing with cards in addition to making new rules. You want that lets you take cards from other players if they fail a dice roll? Just change the rules. You want to be called president and collect tax from other people? Just make a new rule.

This game is played in a correspondence fashion, so you only need to check in every few hours.

Get Started

  1. Create a Discord account if you haven't already.
  2. Ask the person who sent you this invitation for entry to the general channel and introduce yourself! The server admin (me!) will add you to the verified human players list.
  3. After you have been verified, enter the nomic-kittens channel and type !enter to enter as a player and then !help to see a list of all commands.
  4. If this is your first game, you can read a condensed ruling below, or the full rules over at the rules page.
More information on the game: The game will take place in Each turn will have four parts:
  1. You declare a rule to be added, changed or removed.
  2. A vote will be taken, if a majority agrees, it goes through.
  3. Points will be given accordingly.
  4. You play zero or more cards from your hand using !play [card].
    • The play command only checks if you have a card and moves it to the discard, to allow time for a NOPE to go through, an admin will run the effects of the card(s).
  5. Draw a card from the deck using !draw and end your turn using !end.
